Protractor Configuration:
1. Install NPM[nodejs] from google or search "nodejs" on google.
2. On command Prompt install Protractor
   commond : npm install -g protractor
3. Install JDK if it is not in your machine.
4. On command promt install webdriver manager.
   commond : webdriver-manager update
5. Start the webdriver manager from commond promt
   commond : webdriver-manager start.
6. The port address are giver when start the webdriver manager port:444
   check on browser : http://localhost:4444
